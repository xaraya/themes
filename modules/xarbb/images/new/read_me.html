Installation

1. Copying the Images

The folder images should be uncompressed from the distribution archive and then copied to the templates/TEMPLATENAME/images directory. You will most probably be prompted to overwrite the originals.

2. File Changes

No file changes are necessary when using with the FI Subsilver 2.10 theme.

You may need to make minor changes to the index_body.tpl, viewforum_body.tpl & privmsgs_body.tpl files to change the size of the displayed images. If you have followed the instructions given on the Forum Images web site you may need to edit the phpBB theme configuration file and/or templates.
